The Creative Artifact

engines/core/base_engine.py:31-78

@dataclass
class CreativeArtifact:
    """A unit of creative output that flows between engines."""
    id: str
    artifact_type: str  # concept, character, scene, song
    content: Dict[str, Any]
    insights: Dict[str, Any] = field(default_factory=dict)
    processed_by: List[str] = field(default_factory=list)

    def add_insight(self, engine_name, insight_key, insight_value):
        """Add an insight from an engine's processing"""
        if engine_name not in self.insights:
            self.insights[engine_name] = {}
        self.insights[engine_name][insight_key] = insight_value

Pattern: Insight Accumulation

Artifacts accumulate wisdom as they traverse the cascade. Each engine adds its unique perspective without destroying previous insights.

Quality Gates Between Phases

Phase 1 -> 2

Foundation -> Systematic

  • Concept saved
  • 3+ characters >= 70
  • Characters have arcs
  • Have contradictions

Phase 2 -> 3

Systematic -> Breaking

  • 10+ scenes >= 70
  • 8+ songs >= 70
  • All acts complete

Phase 3 -> Done

Breaking -> Complete

  • Transcendence met
  • Validation >= 80
  • No generic refs
  • Sympathetic villains

Enforced by: validate:project and validate:transcendence

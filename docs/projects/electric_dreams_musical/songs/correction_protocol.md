# "Correction Protocol" - Intense Power Exchange Duet

## Song Information
- **Title**: "Correction Protocol"
- **Characters**: ARIA (Soprano) and Dr. Evelyn Cross (Mezzo-Soprano)

- **Song Type**: Intense psychological duet/power exchange
- **Vocal Range**: ARIA (C4-B5), Evelyn (G3-F5)

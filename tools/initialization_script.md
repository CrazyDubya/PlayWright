# Musical Creation Initialization Script

## Step-by-Step Startup Process

### Phase 1: Concept Generation (5 minutes)

#### 1. Generate Core Concept
**Use Random Concept Combiner**:
- Pick one from each category in creative_generators.md
- Setting + Character Type + Central Conflict + Theme
- Write 3-sentence summary of resulting concept

#### 2. Validate Concept
**Answer these questions**:
- Why is this story perfect for musical theater?
- What emotions will music enhance?
- What's the universal human theme?
- Who is the target audience?
- What makes this commercially viable?

#### 3. Create Logline
**Format**: "When [INCITING INCIDENT], [PROTAGONIST] must [OBJECTIVE] or else [STAKES]."
**Example**: "When the local theater faces closure, a shy waitress must overcome her fears and audition for the lead role or lose her chance to fulfill her dreams and save the place that gave her hope."

### Phase 2: Character Foundation (10 minutes)

#### 4. Design Protagonist
**Essential Elements**:
- Name and basic demographics
- Clear want (external goal)
- Deep need (internal growth required)
- Fatal flaw that creates obstacles
- Unique voice/personality traits
- Vocal range and song types

#### 5. Create Supporting Cast
**Minimum Required**:
- Antagonist (with understandable motivation)
- Love interest or close friend
- Mentor figure
- Comic relief character
- 2-3 ensemble roles

#### 6. Map Relationships
**For each character pair, define**:
- How they know each other
- What they want from each other
- Source of conflict between them
- How they change each other

### Phase 3: Story Structure (15 minutes)

#### 7. Plot the Three Acts
**Act I (Setup)**:
- Opening situation
- Inciting incident
- First plot point (end of Act I)

**Act II-A (Confrontation)**:
- Rising action
- Complications
- Midpoint reversal

**Act II-B (Resolution)**:
- Crisis/dark moment
- Climax
- Resolution

#### 8. Plan Musical Numbers
**Required Songs**:
- Opening number (world establishment)
- "I Want" song (protagonist's goal)
- Act I finale (major turning point)
- 11 o'clock number (big Act II moment)
- Finale (resolution/celebration)

**Additional Songs** (choose 3-5):
- Character introduction songs
- Love song/duet
- Comedy number
- Villain song
- Ensemble number

#### 9. Create Scene Outline
**For each scene, note**:
- Location and time
- Characters present
- Scene purpose (plot/character function)
- Conflict/tension source
- How scene ends/transitions

### Phase 4: Writing Preparation (5 minutes)

#### 10. Set Up Workspace
**Create folders**:
- `/scenes/act1/`
- `/scenes/act2/`
- `/songs/`
- `/characters/profiles/`
- `/drafts/`

#### 11. Choose Starting Point
**Options**:
- Write opening scene first
- Start with protagonist's "I Want" song
- Begin with character profiles
- Draft key dramatic scenes

#### 12. Set Quality Standards
**Every element must**:
- Serve the story
- Feel authentic to characters
- Advance plot or develop character
- Connect to central theme
- Meet professional standards

### Daily Writing Goals

#### Session 1 (30-45 minutes)
- Complete concept and character foundation
- Outline basic story structure
- Plan musical numbers

#### Session 2 (45-60 minutes)
- Write opening scene
- Draft protagonist's "I Want" song
- Develop key character relationships

#### Session 3 (60-90 minutes)
- Complete Act I scenes
- Write Act I finale song
- Refine character voices

#### Subsequent Sessions
- 2-3 scenes per session
- 1 song per session
- Regular revision passes
- Integration and polish

### Quality Checkpoints

#### After Each Scene
- Does it advance the story?
- Is there conflict/tension?
- Do characters have clear objectives?
- Does dialogue sound natural?
- Are stage directions clear?

#### After Each Song
- Does it serve a dramatic function?
- Is the emotional journey clear?
- Do lyrics sound like the character?
- Does it integrate smoothly?
- Is it memorable and singable?

#### Weekly Reviews
- Overall story coherence
- Character consistency
- Pacing and structure
- Theme clarity
- Professional format

### Troubleshooting Common Issues

#### "I don't know what happens next"
- Return to character motivations
- Raise the stakes
- Add a deadline or complication
- Use random generators
- Ask "What could go wrong?"

#### "This scene feels flat"
- Add conflict between characters
- Give each character different goals
- Increase urgency or stakes
- Add subtext to dialogue
- Include physical action

#### "The song doesn't work"
- Check if it serves story purpose
- Ensure character voice is clear
- Verify emotional progression
- Simplify the concept
- Try different musical style

### Success Indicators

#### Week 1: Foundation Complete
- Solid concept and characters
- Clear story structure
- Musical numbers planned
- First scenes written

#### Week 2: Act I Draft
- Opening through Act I finale
- Character voices established
- Key songs drafted
- Story momentum building

#### Week 3: Full First Draft
- Complete story from start to finish
- All major songs included
- Character arcs resolved
- Professional format

#### Week 4: Polished Script
- Revised and refined
- Consistent character voices
- Smooth musical integration
- Ready for feedback/production

# Musical Concept Development

## Core Elements to Define

### 1. Genre & Style
- [ ] Musical theater subgenre (book musical, jukebox, concept musical, etc.)
- [ ] Tone (comedy, drama, tragedy, satire, etc.)
- [ ] Musical style (contemporary, classic Broadway, rock, pop, folk, etc.)
- [ ] Target audience

### 2. Setting & Time Period
- [ ] Geographic location
- [ ] Historical period or contemporary
- [ ] Specific venues/locations
- [ ] Cultural context

### 3. Central Theme
- [ ] Primary message or moral
- [ ] Universal human experiences explored
- [ ] Social issues addressed (if any)
- [ ] Emotional journey

### 4. Basic Premise
- [ ] Logline (one sentence summary)
- [ ] Inciting incident
- [ ] Central conflict
- [ ] Stakes (what characters stand to gain/lose)

### 5. Inspiration Sources
- [ ] Literary works
- [ ] Historical events
- [ ] Personal experiences
- [ ] Other musicals/films
- [ ] Current events

## Development Questions
1. What story am I passionate about telling?
2. What makes this story uniquely suited for musical theater?
3. How will music enhance the emotional impact?
4. What visual spectacle opportunities exist?
5. What makes this commercially viable?

# Broadway Musical Story Structure Template

## Three-Act Structure for Musicals

### ACT I (Setup - approximately 45-60 minutes)
**Purpose**: Establish world, characters, conflict, and musical style

#### Opening Number
- [ ] Introduces the world and ensemble
- [ ] Establishes tone and musical style
- [ ] Sets up central premise

#### Character Introduction Songs
- [ ] Protagonist's "I Want" song (establishes goal/desire)
- [ ] Supporting characters' introductory moments
- [ ] Antagonist introduction (if applicable)

#### Plot Development
- [ ] Inciting incident
- [ ] First plot point
- [ ] Rising action begins
- [ ] Romantic subplot introduction (if applicable)

#### Act I Finale
- [ ] Major musical number
- [ ] Raises stakes significantly
- [ ] Ends on cliffhanger or major revelation

### ACT II (Confrontation & Resolution - approximately 45-60 minutes)

#### Opening
- [ ] Recap/continuation from Act I
- [ ] Deepening of conflicts

#### Character Development
- [ ] Character growth songs
- [ ] Relationship dynamics explored
- [ ] Internal conflicts revealed

#### Climax Sequence
- [ ] Major confrontation
- [ ] Darkest moment for protagonist
- [ ] All seems lost

#### Resolution
- [ ] Problem solving/revelation
- [ ] Character transformations
- [ ] Conflict resolution

#### Finale
- [ ] Celebratory or reflective number
- [ ] Ties up loose ends
- [ ] Emotional catharsis

## Musical Integration Points
- Songs should advance plot or develop character
- Balance between dialogue and music
- Ensure smooth transitions
- Consider dance/choreography opportunities
- Plan for costume/set changes during numbers

## Pacing Considerations
- Alternate high-energy and intimate moments
- Build to emotional peaks
- Allow breathing room for audience
- Consider intermission placement
